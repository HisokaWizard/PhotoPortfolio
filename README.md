К тебе обратился знакомый фотограф с просьбой сделать одностраничный сайт-портфолио.

Задача: сделать простое портфолио фотографа. 

Мокап тут: https://1drv.ms/i/s!AgKyCrqi_08vpIp-XiuyDbPEQ1l1Vw

1) Развернуть git-репозиторий на github или bitbucket etc. Все дальнейшие пункты должны фиксироваться коммитами в этот репозиторий(реализовано)
2) Сделать форму для загрузки изображения. Изображение должно отправляться на сервер ajax запросом, без перезагрузки страницы и без фреймов.
Поля формы - "Название изображения" и "Выберите изображение". (реализовано)
3) Изображения складываются в публичную папку, информация об изображениях складывается в базу данных PostgreSQL.
Схема базы на своё усмотрение (не реализовано)
4) Сверстать страницу простым гридом (пример на мокапе). Верстка должна быть адаптивная (реализовано)
5) Реализовать бесконечную подгрузку изображений и добавление их в сетку при прокрутке страницы. (частчно реализовано)
6) Добавить кнопку "Показать случайное изображение", нажатие по которой отправляет ajax запрос на сервер. 
На сервере, с помощью Node.js и PostgreSQL, необходимо достать случайную запись и отдать в колбэк ajax-запроса src изображения и статус,
что всё хорошо в формате json. Показать изображение в модальном окне. (реализовано)
7) С помощью конфига nginx реализовать отдачу изображений с другого субдомена, например static.domain.com.(не реализовано)

Стек технологий:
1) Бекенд: Node.js
2) Фронтенд (на выбор): Pug.js, ES6, HTML5, Sass, Angular
3) Сервер: nginx
4) База данных (на выбор): PSQL, MyS
